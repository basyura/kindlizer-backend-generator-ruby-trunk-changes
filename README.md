kindlizer-backend's generator for ruby-trunk-changes
====================================================

日々更新される [ruby-trunk-changes](http://d.hatena.ne.jp/nagachika) を kindle に配信して勉強するための 
[kindlizer-backend](https://github.com/tdtds/kindlizer-backend) 用ジェネレータ。

kindle.config.yaml
------------------

    :task:
      4:
      - ruby-trunk-changes
